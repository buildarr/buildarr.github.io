v0.7.4 - 2023-12-06#

Initial release of the Buildarr installer for Windows, including the following components:

  • Buildarr Core (v0.7.1)
  • Buildarr Sonarr plugin (v0.6.3)
  • Buildarr Radarr plugin (v0.2.3)
  • Buildarr Prowlarr plugin (v0.5.1)
  • Buildarr Jellyseerr plugin (v0.3.1)
